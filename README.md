# Plant AI Doctor

This repository contains the interim project submission for a GenAI computer vision course.

## Structure
- Notebooks/
  - 01_data_generation.ipynb: Initial exploration of synthetic data generation
  - 02_eda.ipynb: Exploratory data analysis of the real dataset
  - 03_baseline_model.ipynb: Baseline ResNet18 training and evaluation
- data/
  - real/: Real images (chemical / biological)
  - synthetic/: Initial synthetic examples (chemical)
- Presentations/
  - Interim technical presentation (PPT + PDF)
